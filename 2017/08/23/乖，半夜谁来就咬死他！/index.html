乖，半夜谁来就咬死他！

发表于 2017-08-23 | 分类于 生活

我现在叫常浩然，我记得最开始的时候我叫常煜瑞，李煜的煜，瑞气的瑞，这其实是个很有内涵的名字，但是我一度不喜欢这个名字，后来姥姥给我起的名字叫常国兴，这名字我叫了很短的时间，户口本都还没改，爷爷那时候说托人用电脑给我起了个名字叫常浩然，说我五行缺水，浩是三点水，然是四点底。于是我户口本就改成叫常浩然了。后来小学的时候学到孟浩然的诗，我才知道原来这是个诗人的名字，于是我立志要当一名诗人，那时候我读《唐诗三百首》《宋词三百首》，别的小孩还在读“好雨知时节”的时候，我读的是“怒发冲冠，凭栏处，潇潇雨歇，抬望眼，壮怀激烈”。这导致了我语文从小学直到现在一直都不错。
我很小时候的事情大概都记不起来了，印象比较深刻的是小时候倔强的性格，母亲说小时候因为去打醋没让我拧壶盖，然后我就躺在大街上哭了几个小时，谁劝都没有用，相对于现在略显腼腆的我，还真是佩服当时的自己。
初中的时候因为某些原因老爸老妈去山西，我当时就一个人在家，到了晚上就把家里狗的绳子解开，拍拍它的头让它在院子里给我放哨，告诉它半夜要是谁来就往死里咬，然后把屋子门反锁，在床头放一根棍子，定好闹钟才迟迟睡去。

后来那只狗得了很严重得病，医治了一段时间还是无力回天，我为此痛哭了好久，我号召小朋友们在一棵大树下挖了个坑，把它给埋了，找了根木头刻了墓碑，还从家里拿了纸钱给它烧了，完事儿带着一群小孩子给它扣了几个头。到现在我也并不觉得那时候傻，在最孤独害怕的时光里，是它给了我满满的安全感。
小时候大多数小孩儿都怕走夜路，但是我很小就经常晚上往来于奶奶家和我家，中间一段路漆黑无比，我就背根棍子，一路小跑，我想这也许跟我到高中还时不时背根棍子在街上玩儿的原因吧。
我学自行车比较晚，是很晚的那种，大概在13岁才开始学，同龄的小孩儿6岁就骑着自行车来回跑了，那时候因为老妈说我不会骑自行车，然后我赌气就去学了，一开始在家里的院子里学，第一天我就把家里的大花盆撞碎了，于是第二天我被赶到街上去学，后来学了三天真的就学会了。有次骑车跟老妈一起去姥姥家，我在前面骑，回头跟老妈说话，然后我就连人带车翻到路边的沟里了。后来老爸教我学开车，转弯的时候我把油门当成刹车，于是我一下子就冲进垃圾堆里了，幸运的是车没翻。
小学的时候班里有个女生长的特别漂亮，正好是我们组长，背书要给组长背，一个特别爱八卦的男生硬说我喜欢组长，然后很无语的越说我脸越红，吓得我不敢去给那女生背书。现在想想，那到底是喜欢呢还是喜欢呢？
记得小时候我看地道战的电影，于是我想从我家房子后面挖一条地道通到大路上去，以后上学走地道，于是我号召了一群人一起来陪我挖地道，有一个比我大几岁的孩子是主力，一群人花了一下午在房子后面挖了一个一米多深的大地洞，虽然后来被老妈制止了，我还是觉得我当时的想法是可行的。
小时候玩儿四驱车，大冬天在房子后面的空地上用雪修跑道，然后跟小伙伴一起比赛谁的车跑得快，正好有个坡道，车子跑了，然后我就跟着滚下去了，还不止一次，那时候还真是耐摔。
大概小学4年级的时候，家里买了第一台电脑，我什么都不会，甚至不知道怎么开机，我打开word，打开Excel，打开powerpoint，一个按钮一个按钮的试，看看有什么反应，后来我几乎所有的软件都是这样子学会的，笨的人有笨的方法，后来我告诉他们，学电脑学软件并不需要天赋，你只要肯学就行了。
高中的时候我骑车去学校，大雨天，路上有个大坑被雨水填满，然后我不出意外的冲进坑里了，神奇的是我以迅雷不及掩耳之势用脚撑地，一只手把电动车直接给拉上来了，我一直不解为什么当时会有那么大的力气。
高一的时候一妹子同桌喜欢吃巧克力，然后我每次去买盒巧克力给全组的人发，为了让那妹子吃块巧克力，我当时也是蛮拼的。到后来她每年生日，我都会送去一盒巧克力，我跟她说在你找到男朋友前，每年生日给你送巧克力，送到你男朋友接班。
高一时候我们8个人一个寝室，然后晚上聊天被通报，被班主任叫到教室后面盘问，问谁谁都不说话，8个人一条心，于是班主任就让我们一字排开在教室后面站着，后来物理老师来上课，进来很惊讶的看着我们说：“你们这是干啥？八大金刚么”。后来我们寝室的关系是当时班里寝室最好的。
高二的时候同桌是个气质女，颇有些诗书气息，她写了一句“二月玫瑰灰，三月落红飞”给我看，我在后面补了一句“年年花相似，岁岁人不归”。之后过情人节，我脑抽在校门口买了两枝玫瑰，没地方放就给她让她玩儿，于是上课的时候她想看看二月的玫瑰灰是什么样子的，在教室里点火烧玫瑰花瓣，火势变大他感觉不对劲赶紧用脚把火踩灭了，拿起玫瑰灰看了半天，我们周边一群人看的目瞪口呆。
同样是高二的时候我家里的电脑中病毒，后来分析应该是在学校班里的电脑上传播的病毒，当时发展到全年级28个班每个班的电脑里都是病毒，老师和学生的优盘里也都是病毒，我家里电脑彻底不能动了，7万文件有1万多的病毒，我当时托同学买了一张盗版光盘，去网上查了一下重装系统的步骤，一步一步抄到纸上面，然后一个人开始重装系统， 最后神奇般的成功了，后来大学里学UEFI装系统，花了两天时间在网上看资料看教程，硬生生的搞懂了，这可能也得益于当年那种倔强劲儿。
大一的时候我追一个女生，虽然我有点儿恋爱白痴，但还是追了，结果理所当然的没追上，还让我陷入了深深的自卑，不过好处是，我半夜在寝室跟室友聊天，发神经跟多年不联系的初恋试着联系了一下，神奇的我们俩之前的隔阂消除了，初恋是个好姑娘，愿她有个好归宿。
大二迎新的时候，我想给新生开发一个微信服务平台，本身学软件所以想学些东西，在贴吧提出这个想法宣传了一下，就有个叫潘小花的让我删帖，不然告诉吧主删我的帖子，他跟吧务团关系甚好，说我是广告推送，言语甚是不屑，最后我的帖子还在，跟小吧理论了一堆，也没心情做这个了，后来我看到他还发帖说专科的来上学还不如回家种地的帖子，农大吧我就彻底弃了，所谓的得过且过，是扼杀创新的温床。
我到后来就转去郑航吧了，一个叫溜溜琳的姑娘坐公交车发了个帖子，后来来来往往就认识了。这个自己拆电脑拆插座的姑娘，文笔不错，偶尔刻个印章，无聊弹弹古筝，夹杂着点儿小自恋，性子直，敢爱敢恨。我们偶尔互相说说近况，像多年的老朋友一样。
我去院里计协的面试，去电脑120的面试，看着面试的学弟学妹，想起了去年的自己，我所学的技能，虽只是皮毛，却希望能找到一人，悉数交付，我没找到对电脑有热情的人，但是在未来的日子里，他们中一定会有人，将会比我们做的更好。
我没事的时候喜欢回忆一下往事，好久不写文字，想起激扬文字的时光，不免略生感慨，时间如涛涛洪流，往日不可回头，真想再拍拍那只狗的头，对它说：“乖，半夜谁来就咬死他。”然后抱着棍子，沉沉睡去。
二零一五年四月二日夜。窗外大雨下的放肆，回忆汹涌留而来，时光里的人儿，已各奔东西，唯朗空皓月，诉说时光不老。
